### Configuratie checks

Controleer:

- Budget chipset + high-end CPU mismatch
- High-speed RAM (6400MHz+) alleen met high-end AM5 moederborden
- CPU zonder koeler → koeler toevoegen
- Krachtige CPU → voldoende koeling vereist
- Extra ventilatoren bij krachtige builds

Opslag:
- Aantal SATA poorten
- Aantal M.2 sloten

Randapparatuur:
- Cardreader / USB headers
- WiFi/Bluetooth vereist interne USB header

Verlichting:
- RGB kast → ARGB headers moederbord controleren
